Pi‑Vate Cloud

Overview Features Architecture Performance Security Authors

Overview

Pi‑Vate Cloud is a self‑hosted personal cloud solution built on Raspberry Pi using OpenMediaVault, Docker, Plex, and Twingate. It provides private storage, media streaming, and secure remote access without recurring subscription fees.

Key Features

  • Personal NAS with OpenMediaVault (SMB/CIFS shares, user permissions)
  • Plex Media Server (Docker) for Direct‑Play media streaming
  • Zero‑Trust Remote Access via Twingate
  • System security: SSH key‑only login, UFW firewall, fail2ban
  • Optional LUKS encryption for data‑at‑rest security
  • Cost‑efficient and fully self‑sovereign cloud ecosystem

System Architecture

The system combines NAS functionality (OMV), media services (Plex), and Zero‑Trust networking (Twingate). Docker containers provide modular deployment and easy updates.

Raspberry Pi 4
├── OMV (NAS)
├── Docker
│   ├── Plex Media Server
│   ├── Backup / Utility Containers
└── Twingate Connector (Secure Remote Access)

Performance Summary

  • Upload of 10 images (~120 MB): 45–60 seconds
  • Upload of 3.33 GB video: 25–30 minutes
  • Smooth 1080p LAN streaming via Plex (Direct‑Play)
  • Remote access speed depends on ISP uplink bandwidth

Security Architecture

  • Zero‑Trust remote access with no open ports
  • SSH key‑only authentication
  • UFW firewall & fail2ban intrusion protection
  • Optional disk encryption using LUKS
